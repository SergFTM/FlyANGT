### ДИЗАЙН-СИСТЕМА (строго соблюдать)

**Палитра:**
```
--color-bg-dark: #0a1628        /* тёмно-синий, основной фон hero */
--color-bg-light: #f7f4ef       /* тёплый слоновая кость, секции */
--color-bg-white: #ffffff        /* карточки */
--color-gold: #c6a36b           /* акцент, заголовки, CTA */
--color-gold-dim: rgba(198,163,107,0.4)
--color-text-dark: #1a2a3a      /* заголовки на светлом */
--color-text-body: #555555      /* основной текст */
--color-text-muted: #999999     /* подписи */
--color-text-light: rgba(255,255,255,0.75) /* текст на тёмном */
```

**Типографика (Google Fonts):**
```
Display / H1-H2:   'Playfair Display', serif — weight 400–700, italic для акцентов
Body / описания:   'Cormorant Garamond', serif — weight 300–500, italic
UI / метки / CTA:  'Montserrat', sans-serif — weight 200–700
```

Правила:
- Метки (labels) — Montserrat 9–10px, letter-spacing 3–5px, uppercase, приглушённый цвет
- Заголовки — Playfair Display, крупно, с выраженной иерархией
- Тело текста — Cormorant Garamond, 16–18px, line-height 1.7, не больше 60 символов в строке
- Никаких Inter, Roboto, Arial
- Золотые акценты — экономно, только для самого важного

**Визуальный стиль:**
- Журнальная элегантность: много воздуха, утончённые линии-разделители (1px, золото или rgba)
- Тёмные секции чередуются со светлыми (dark → light → dark)
- Фотографии самолёта — полноширинные, с градиентным overlay
- Анимации: плавный fade-in при скролле (Intersection Observer), staggered reveal карточек
- Никаких дешёвых теней, градиентных кнопок, неоновых акцентов
- Парящие элементы: тонкие золотые линии, аккуратные иконки ✦

### ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ

**SvelteKit:**
- Svelte 5 с runes (`$state`, `$derived`, `$effect`)
- Scoped стили в `<style>` каждого компонента
- Компонентная структура: `Hero.svelte`, `Advantages.svelte`, `Safety.svelte`, `CyprusRoutes.svelte`, `Comparison.svelte`, `Specs.svelte`, `CTA.svelte`, `Footer.svelte`
- Главная страница: `+page.svelte` импортирует все секции

**Анимации:**
- Intersection Observer для fade-in при скролле (написать action `use:reveal`)
- Staggered reveal для карточек (CSS `animation-delay`)
- Smooth scroll для якорных ссылок
- Parallax-эффект на hero-фоне (лёгкий, через `transform: translateY`)
- Hover на карточках: translate -2px, тень усиливается

**Изображения:**
- Плейсхолдеры: использовать `<div>` с gradient-фоном в стиле палитры
- Комментарии `<!-- REPLACE: фото самолёта в полёте -->` где нужны реальные фото
- Оптимизация: `loading="lazy"`, srcset если есть

**Адаптив:**
- Mobile-first: от 320px
- Breakpoints: 768px (tablet), 1024px (desktop), 1440px (wide)
- На мобильных: карточки в столбец, hero-текст меньше, маршруты в 1 колонку

**SEO:**
- Мета-теги: title, description, og:image
- Семантический HTML: header, main, section, footer
- Все изображения с alt-текстом
